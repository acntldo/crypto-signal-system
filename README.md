# 📊 Crypto Signal Intelligence System
An AI-powered crypto trading signal dashboard using machine learning, FastAPI, and React.
The system analyzes historical market data and generates BUY / HOLD / SELL signals with a real-time interactive UI.

---

## 🧠 Key Features
- ML-based trading signal prediction
- Technical indicators (MA, RSI, momentum, volatility)
- Interactive React dashboard
- FastAPI backend for inference
- Real-time market visualization
- System telemetry panel

---

## 🏗️ Tech Stack
- React
- FastAPI
- Python (scikit-learn, pandas)
- Recharts
